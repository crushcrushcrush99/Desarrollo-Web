# POO en Typescript

### Comandos usados en aula
    -npm install -g typescript [Instalar typescript] 
    tsc --init [inicializa as configuracoes do typescript]
    tsc --watch [transpila arquivos ts para js]
    rm - remove arquivos
    rm - rf- remove pastas
    rm *.js(mover a una carpeta todo lo js)
